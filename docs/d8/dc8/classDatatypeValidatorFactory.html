void DatatypeValidatorFactory::expandRegistryToFullSchemaSet ( )
private

Initializes registry with primitive and derived Simple types.

This method does not clear the registry to clear the registry you have to call resetRegistry.

The net effect of this method is to start with the smallest set of datatypes needed by the validator.

If we start with Schema's then we initialize to full set of validators.
