Detailed Description

This class provides a simple means to track ID Ref usage. Since id/idref semantics are part of XML 1.0, any validator will likely to be able to track them. Instances of this class represent a reference and two markers, one for its being declared and another for its being used. When the document is done, one can look at each instance and, if used but not declared, its an error.

The getKey() method allows it to support keyed collection semantics. It returns the referenced name, so these objects will be stored via the hash of the name. This name will either be a standard QName if namespaces are not enabled/supported by the validator, or it will be in the form {url}name if namespace processing is enabled.
