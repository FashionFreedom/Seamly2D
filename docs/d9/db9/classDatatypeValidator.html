Detailed Description

DataTypeValidator defines the interface that data type validators must obey. These validators can be supplied by the application writer and may be useful as standalone code as well as plugins to the validator architecture.

Notice: The datatype validator will own the facets hashtable passed to it during construction, which means that the datatype validator will be responsible for the deletion. The facets hashtable will be created during parsing and passed to the appropriate datatype validator which in turn will delete it upon its destruction.
