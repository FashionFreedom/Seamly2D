34 //
35 // Define inline casting functions to convert from
36 // (DOMNode*) to the embedded instances of DOMNodeImpl,
37 // DOMParentNode, and DOMChildNode.
38 //
39 // Each type of embedded object corresponds to a HasXXX virtual
40 // interface class that a given DOM implementation class will
41 // support to expose its embedded object(s) to other implementation
42 // classes.
43 //
44 // This replaces the previous implementation that relied upon unsafe
45 // casts and member offsets that rely on unspecified behavior in C++,
46 // with a hopefully small cost in memory and performance.
47 //
