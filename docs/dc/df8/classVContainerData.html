Detailed Description

A data container class that uses Qt's implicit sharing mechanism.

The VContainerData class is designed to efficiently manage shared data using Qt's implicit sharing (copy-on-write) mechanism. It contains several hash maps and shared pointers to handle various elements.

The class inherits from QSharedData to leverage Qt's implicit sharing. It has two constructors: one for initializing its member variables and another for copying the contents from another instance. The class manages the following members:

  • gObjects: A hash map mapping a quint32 to a shared pointer of VGObject.
  • variables: A hash map mapping a QString to a shared pointer of VInternalVariable.
  • pieces: A shared pointer to a hash map mapping a quint32 to VPiece.
  • piecePaths: A shared pointer to a hash map mapping a quint32 to VPiecePath.
  • trVars: A pointer to a VTranslateVars object.
  • patternUnit: A pointer to a Unit object.

The class provides a constructor for initializing these members and a copy constructor to create copies of the data container.
