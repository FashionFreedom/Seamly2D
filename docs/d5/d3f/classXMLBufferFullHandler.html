Detailed Description

XMLBufferFullHandler is a callback interface for clients of XMLBuffers that impose a size restriction (e.g. XMLScanner). Note that this is intended solely as a mix-in for internal use, and therefore does not derive from XMemory (to avoid the ambiguous base class problem).
