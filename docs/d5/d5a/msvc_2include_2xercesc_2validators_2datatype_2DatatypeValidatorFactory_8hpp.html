This class implements a factory of Datatype Validators. Internally the DatatypeValidators are kept in a registry. There is one instance of DatatypeValidatorFactory per Parser. There is one datatype Registry per instance of DatatypeValidatorFactory, such registry is first allocated with the number DatatypeValidators needed. e.g. If Parser finds an XML document with a DTD, a registry of DTD validators (only 9 validators) get initialized in the registry. The initialization process consist of instantiating the Datatype and facets and registering the Datatype into registry table. This implementation uses a Hashtable as a registry. The datatype validators created by the factory will be deleted by the registry.

As the Parser parses an instance document it knows if validation needs to be checked. If no validation is necessary we should not instantiate a DatatypeValidatorFactory. If validation is needed, we need to instantiate a DatatypeValidatorFactory.
