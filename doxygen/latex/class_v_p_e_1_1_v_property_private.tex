{\footnotesize\ttfamily QVariant VPE\+::\+VProperty\+Private\+::\+Variant\+Value}

The property\textquotesingle{}s value. This does not have to be used by subclasses, but it makes sense in cases where QVariant supports the data type. Also, this can be used as cache, so that when the data() function gets called by the model, the data does not have to be converted in a QVariant every time.
