\doxysubsection{Detailed Description}
Class showing a widget as popup window. 

set\+Widget() function allows you to specify the widget to be popped up. After widget is set, you normally should call show() slot in order to pop the widget up at the specified global position.

\mbox{\hyperlink{class_v_widget_popup}{VWidget\+Popup}} takes care about positioning of your widget on the screen so it will be always visible even if popped beside.
