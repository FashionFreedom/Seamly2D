\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename TBase, typename TString$>$\newline
class qmu\+::\+Qmu\+Parser\+Token$<$ TBase, TString $>$}

Encapsulation of the data for a single formula token. 

Formula token implementation. Part of the Math Parser Package. Formula tokens can be either one of the following\+: 
\begin{DoxyItemize}
\item value 
\item variable 
\item function with numerical arguments 
\item functions with a string as argument 
\item prefix operators 
\item infix operators 
\item binary operator 
\end{DoxyItemize}
