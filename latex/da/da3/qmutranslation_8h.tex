\doxysubsection{Detailed Description}
The VTranslation class help store string for translation. 

I took idea from this article \href{http://ololoepepe.blogspot.com/2013/08/qt.html}{\texttt{ http\+://ololoepepe.\+blogspot.\+com/2013/08/qt.\+html}}. As you know, if wrap string to a function translate, it will be marked for translation. No matter what namespace contains this function. In class Translation used this circumstance. This mean never change name of method translate!!!!!. Instead of using QT\+\_\+\+TRANSLATE\+\_\+\+NOOP3 macros we can store strings in \mbox{\hyperlink{classQMap}{QMap}}.
